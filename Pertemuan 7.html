Web Lirik — Mardua Holong (template)

Template menampilkan lirik, karaoke sederhana, dan fitur ekspor. Tidak menyertakan lirik asli.
Batak
Template siap disunting
Audio (opsional) — pilih file .mp3/.ogg
Kecepatan baris (ms):
1200 ms
Petunjuk singkat: tempel lirik -> klik "Terapkan lirik". Untuk menyinkronkan timestamp: upload audio, putar audio, lalu klik tiap baris saat audio mencapai baris itu (gunakan tombol "Rekam Timestamp (manual)"). Setelah selesai klik "Ekspor .lrc" untuk mendownload file lirik dengan waktu.
